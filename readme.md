## `markdown`图片助手

![image](logo.png)

- 截图后直接粘帖到文本框
- 图片上传完成后自动加入剪贴板
- 点击图片复制 `markdown` 格式图片
- 需要配置[腾讯云存储参数](https://console.cloud.tencent.com/cos)

![image](http://img.s1.lisz.vip/2019-5-8/s7eofkfk4hj.png)

| 来源 Origin | 操作 Methods | Allow-Headers | Expose-Headers | 超时 Max-Age |
|-- | -- | -- | -- | -- |
| * | PUT,GET,POST | * | ETag | 30 |
